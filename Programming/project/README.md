# ShopClone — Static E‑commerce Clone

This is a small static clone demo built with plain HTML, CSS and JavaScript. It includes a product grid, search box and a client-side cart drawer.

Files included:

- shop.html — main demo page
- styles.css — styles used by the demo
- script.js — cart and search interactivity

To view locally:

Option A — open in browser:

1. Open `shop.html` in your browser.

Option B — run a simple local server (recommended):

PowerShell / Terminal:

```powershell
python -m http.server 8000

# then open http://localhost:8000/shop.html in your browser
```

Notes:

- This is a static frontend demo — no backend or payments.
- Images use placeholder URLs; swap them for local assets in an `assets/` folder if desired.
